## 📌 What is FRP (Factory Reset Protection)?

**Factory Reset Protection (FRP)** is a built-in Android security feature that activates automatically when a Google account is registered on a device. If a factory reset is performed outside of the Settings menu, the device will require the previously registered Google account credentials before it can be used again.

This is designed to protect against theft — but it can lock out legitimate owners who have forgotten their credentials or purchased a second-hand device.
